=head1 NAME

dts - study C<developmental time> (similarity matrices)

=head1 DESCRIPTION

Study C<developmental time> similarity matrices generated from twin networks. This command-line tool invokes sub-commands, as many linux tools do.

=head1 USAGE

 dts '<help|usage|man>'

=head1 SYNOPSIS

       $ dts usage
       $ dts man

=head1 dts sessions

List available, installed sessions.

=head2 SESSIONS

These are configurations, which alter the environment dts is running.

Reserved: L<installation>, L<debug>.
