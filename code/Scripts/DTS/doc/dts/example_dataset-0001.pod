=head1 EXAMPLE DATASET (0001)

A select dataset to test/validate this package.

=head1 CONTENTS

Following datasets are provided: zebrafish1, medakashort1, stickleback1, celegans1.

=head1 SYNPOSIS

  # download the complete dataset
  $ dts call system/objects/example-dataset/download system/contexts/example-dataset

  # or
 
  # download the zebrafish1 sub-dataset
  $ dts call system/objects/example-dataset/download system/contexts/example-dataset/zebrafish1

  # (re)build "zebrafish1" in example-dataset/
  $ dts call system/objects/example-dataset/rebuild system/contexts/example-dataset/zebrafish1

  # rebuild full example-dataset (may take rather long)
  $ dts invoke objects/example-dataset/rebuild contexts/example-dataset/*1

  # open app1 (ubuntu)
  $ dts invoke objects/app1/ui_open_qmlscene contexts/example-dataset/zebrafish1

  # open app1 (macos)
  $ dts invoke objects/app1/ui_open_qml contexts/example-dataset/zebrafish1

=head1 INVOCATION

=head2 dts objects

List available, installed objects. This package object are:

 example-dataset  dowload, build, update the example dataset

=head3 Examples

Generate plots from subset of dataset, and cache them as invidiual png files. These are later used by the interactive UI.

  # Single file

  $ dts call system/objects/example-dataset/build_plot_write_to_disk system/contexts/example-dataset/zebrafish1/parametersweep1/pset1 data/example-data/zebrafish1/unified/MATs/TN_*.mat

  # Example: using multi-core system for batch processing (using the parallel[1] tool)

  $ find data/example-data/zebrafish1/unified/MATs/TN_*.mat | parallel dts call system/objects/example-dataset/build_plot_write_to_disk system/contexts/example-dataset/zebrafish1/parametersweep1/pset1 {}

=head1 COMPRESS INTO .zip

  # Will write into cache/compressed/ as a zip file
  $ dts call system/objects/example-dataset/compress/ system/contexts/example-dataset commit

  # or only the zebrafish1 subset
  $ dts call system/objects/example-dataset/compress/ system/contexts/example-dataset/zebrafish1 commit

=head1 UNCOMPRESS FROM .zip (once downloaded, or compressed previously)

  # review
  $ dts call system/objects/example-dataset/uncompress system/contexts/example-dataset/medakashort1 

  # commit
  $ dts call system/objects/example-dataset/uncompress system/contexts/example-dataset/medakashort1 commit
